## NewTon DC Tournament Manager v4.0.16 Release Notes

**Release Date:** November 23, 2025
**Incremental Release**

---

## Overview

**NewTon DC Tournament Manager Version 4.0.16** adds a clock to the main header, providing consistent time visibility across all pages.

This release contains no breaking changes and is a drop-in replacement for v4.0.15.

**Key Highlight:**
- Clock visible on all pages, matching Match Controls styling

---

## ‚ú® New Feature

**Header Clock:**

Added a clock to the upper right corner of the main header, aligned with the title and logo.

**What Was Added:**
- Clock displayed in header on all pages (Setup, Registration, Tournament, Config)
- Styled identically to Match Controls clock (border, shadow, monospace font)
- Updates every 10 seconds

**Why It Matters:**
Operators can now see the current time at a glance without opening Match Controls, useful during tournament setup and player registration phases.
